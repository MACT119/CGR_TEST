SIMULADOR CGR (OFFLINE) - DEMO

Qué es
- Aplicativo web tipo test que funciona SIN servidor: abre index.html en el navegador.
- Banco de preguntas de ejemplo incluido (NO oficial).
- Permite importar un questions.json propio para reemplazar el banco.

Cómo abrir en celular (sin servidor)
1) Copia toda la carpeta "CGR_TestOffline" al celular (o a Drive/WhatsApp y descárgala).
2) Abre "index.html" con tu navegador (Chrome/Edge/Safari).
   - En Android: suele funcionar abriendo desde el gestor de archivos.
   - En iOS: funciona mejor guardando la carpeta en "Archivos" y abriendo index.html.

Importar banco de preguntas
- En la pantalla inicial, pulsa "Importar preguntas (JSON)" y selecciona tu archivo.
- Formato del JSON: ver questions.sample.json

Exportar progreso
- Botón "Exportar" (arriba) descarga un archivo JSON con tu progreso y algunos ajustes.

Nota legal/ética
- Este simulador no debe contener ni divulgar preguntas oficiales del concurso.
  El contenido del banco debe ser ORIGINAL (práctica).
